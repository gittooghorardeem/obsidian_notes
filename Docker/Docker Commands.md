- `docker --version`
- `docker --help`
- `docker run --help`
- `docker search <Name of the image>` - Searching for docker images
- `docker pull <name of the image>`
- `docker images` > Lists the current downloaded images in the docker system
- `docker ps` > lists the currently run containers in the system
- `docker run <image id>` > Start the docker image installed in the system
