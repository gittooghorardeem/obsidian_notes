1. **What is Playwright?** : 
	1. *Playwright is an open-source end-to-end testing and browser automation library (created by Microsoft) that lets you script and test web apps across Chromium, WebKit and Firefox via a single, modern API. Itâ€™s designed for reliability (auto-waiting, context isolation) and modern web features (single-page apps, WebSockets, etc.).*
2. **Which programming languages does Playwright support?**
	1. *JavaScript/TypeScript, Python, Java, and C#.*
3. **Which browsers are supported by Playwright?**
	1. *Chromium (Chrome/Edge), WebKit (Safari engine), and Firefox.*
4. **How is Playwright different from Selenium?**
	1. *Modern APIs vs older WebDriver protocol.*
	2. *Auto-waiting built-in (less flakiness).*
	3. *Multiple browsers with unified API and browser contexts.*
	4. *Faster and more deterministic in many scenarios.*
5. **What is a Browser Context in Playwright?**
	1. *A lightweight, isolated environment inside a browser instance (like an incognito window): separate cookies, localStorage, cache.*
6. **What is auto-waiting in Playwright?**
	1. *Playwright automatically waits for elements to be actionable (visible, attached, stable) before performing actions like click/type.*
7.  **What is page in Playwright?**
	1. *`page` represents a single browser tab/frame context used to navigate and interact with the DOM, network and events.*
8. **How do you install Playwright?**
	1. `npm init playwright@latest`

	3. `npx playwright install`
