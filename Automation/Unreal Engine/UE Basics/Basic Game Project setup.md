> **Launched**

- Selecting the Blank Game Project
	- Set to *Blueprint*  *(minimum coding)*
- Selecting Blank Game Project
	- use C++
		- This will trigger to install Windows VS 2022 (if not already installed)
		- After installing VS 2022
		- Launch UE editor, **Go to > Tools > Open in VS 2022**
		- To change the VS code Goto UE editor **Edit > Editor preferences > search "Code editor" > select Visual Studio Code** 
		- Then Go back to **Tools > Refresh Code**
- Change Layout : 
	- UE editor , **Go to Windows > Load Layout > UE 4** for the current project

> **UE launched UI dashboard**

- 4 main panels
	- Viewport (*opens by default*)
	- Content Browser (*windows > content Browse*)
	- Outliner (*all the objects placed in the the viewport*)
	- Details (*loads all the attributes and properties of the selected object*)

> **Settings**

- Game World : *Windows > World Settings*
- Multi View Port : *Windows > Viewports*
- New C++ class : *Tools*
- Build settings : Building lighting and stuff, baking the lighting
- Show FPS : 
	- "~" to focus on the command line
	- type : `stat fps` : show the fps on the viewport
	- type : `t.MAXFPS 999` : unlock (kind of) FPS limiter
- `F` - Focus moves to the selected element 
- `RMB + WASD` - free rotate in the Viewport
- **Local Space Coordinates**
- **Global Space Coordinate**
- `END` will snap the selected object to the ground
