**Prerequisite :** 
1. Install Node.js
2. npm should be bundled with node.js installation 
3. if npm is not found (windows 11) run the following in windows Powershell with Admin access
	1. `Set-ExecutionPolicy Unrestricted`
4. Install a Code editor. (in my case i used Visual Code or VS code)

**Getting started with Cypress Tutorials**

5. Clone Project (Tutorials basis)
	1. `git clone <repository>`
6. (Used VS code personally)
7. `npm install` (This will install the dependencies)
8. `npm install cypress --save-dev`
9. `npx cypress open` (Configures cypress)
10. `Spec` files aka test files
11. following up, this will create a "cypress" folder in the vs code project
